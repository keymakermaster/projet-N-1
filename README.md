# Logo Factory

A simple web application for creating and customizing logos. Built with Next.js and `lucide-react`.

## Features

- Select from hundreds of icons.
- Customize icon size, color, and rotation.
- Customize background shape, color, and padding.
- Live preview of the logo.
- Download as SVG, PNG, or ICO.

## Running the development server

Inside the `my-logo-app` directory, run:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
